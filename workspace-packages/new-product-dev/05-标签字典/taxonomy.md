# 标签字典 — VOC 九维 + 十三种需求

> 所有打标必须严格使用以下维度与名称，不得自创。源自主站 VOC 标签体系（TAG_WORKERS）。

## 一、VOC 九维（每条评论逐维判定）

| # | 维度 | 说明 | 常见标签示例 |
|---|---|---|---|
| 0 | 用户画像 | 人口统计 + 心理特征 | 送礼人群 / 老用户 / 租房党 / 养宠家庭 |
| 1 | 购买动机 | 功能驱动 + 情感驱动 | 性价比 / 情感送礼 / 功能需求 |
| 2 | 购买目的 | 使用目标 | 节日赠礼 / 自用 / 企业采购 |
| 3 | 使用场景 | 何时何地用 | 沐浴场景 / 差旅 / 客厅摆放 |
| 4 | 使用阻碍 | 痛点 / 卡点 | 吸力不足 / 噪音大 / 安装复杂 |
| 5 | 克服方式 | 用户如何自救 | 加静音棉 / DIY 改装 / 自购配件 |
| 6 | 用户忠诚度 | 复购 / 流失 | 忠实复购 / 流失风险 |
| 7 | 产品改进建议 | 用户给的方案 | 增强吸盘 / 出便携版 / 快装结构 |
| 8 | 十三种需求 | 深层驱动（见下） | 安全 / 舒适 / 审美 / 价格 |

> **否定保护**：含否定词的表述（如「不噪音大」「没有不好」）不计入对应负面维度。

## 二、十三种需求 · 四层模型（维度 8 的取值层级）

自下而上（越下层越「必买」，越上层越「身份认同」）：

1. **必买层**：安全、健康、价格 —— 不满足则绝不考虑。
2. **功能层**：功能、效率、可靠 —— 核心使用价值。
3. **体验层**：舒适、便利、审美 —— 使用愉悦度。
4. **身份层**：社交、尊重、自我实现 —— 情感与身份认同。

### 常用需求词表
- 安全 / 健康 / 价格（必买）
- 功能 / 效率 / 可靠（功能）
- 舒适 / 便利 / 审美（体验）
- 社交 / 尊重 / 自我实现（身份）

## 三、使用约定
- 一条评论可命中多个维度，每个维度可多个标签。
- 维度 8 的取值须落在上述四层词表内。
- 打标结果与行号绑定，供需求聚类与四象限复用。
